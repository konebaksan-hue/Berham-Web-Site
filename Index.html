Урок 0. Ознакомление с Blender

Blender — бесплатная кроссплатформенная программа с открытым исходным кодом для 3D‑моделирования, анимации, рендеринга, видеомонтажа и многого другого. Используется как любителями, так и профессионалами (в т. ч. студиями для создания полнометражных анимационных фильмов). Системные требования Минимальные: процессор: 64‑битный четырёхъядерный CPU; ОЗУ: 8 ГБ; видеокарта: OpenGL 4.3 с 2 ГБ VRAM; место на диске: 500 МБ. Рекомендуемые: более мощный процессор и видеокарта для комфортной работы с большими сценами. Установка Перейдите на официальный сайт Blender (blender.org). Скачайте последнюю версию для вашей ОС. Запустите установщик и следуйте инструкциям. После установки запустите программу. Урок 2. Интерфейс и навигация Основные элементы интерфейса: 3D Viewport — основное окно для работы с объектами. Outliner — список всех объектов сцены. Properties — панель настроек объектов, материалов, рендера и т. д. Timeline — временная шкала для анимации. Topbar — главное меню и рабочие пространства. Навигация в 3D Viewport: Вращение камеры: зажмите среднюю кнопку мыши (колёсико) и двигайте мышь. Приближение/отдаление: прокрутка колёсика мыши. Перемещение по сцене: Shift + средняя кнопка мыши. Фокус на объекте: выделите объект и нажмите . (точка на нумпаде). Урок 3. Базовое моделирование Создание объектов: Нажмите Shift + A, чтобы открыть меню добавления объектов. Выберите тип объекта (например, Mesh → Cube). Новый объект появится в центре сцены. Режимы работы: Object Mode — работа с объектами как целыми сущностями (перемещение, масштабирование, вращение). Edit Mode — редактирование геометрии объекта (вершины, рёбра, грани). Переключение: Tab. Базовые инструменты: Перемещение: G + ось (X, Y, Z). Вращение: R + ось. Масштабирование: S + ось. Выдавливание (Extrude): E — позволяет «вытянуть» грань или вершину. Разрезание (Loop Cut): Ctrl + R — добавляет новые рёбра для детализации. Практическое задание: создайте простую сцену с домом из кубов и пирамидой (крыша). Используйте G, R, S для позиционирования. Урок 4. Материалы и текстуры Назначение материала: Выделите объект. Перейдите во вкладку Material Properties (иконка сферы). Нажмите New для создания нового материала. Настройте цвет, отражение, шероховатость и т. д. Добавление текстуры: В настройках материала найдите раздел Base Color. Нажмите на кружок рядом с параметром и выберите Image Texture. Загрузите изображение текстуры. В Edit Mode разверните UV‑карту объекта (U → Unwrap), чтобы текстура правильно легла на поверхность. Урок 5. Освещение и рендеринг Типы источников света: Point Light — точечный источник (лампочка). Sun Light — направленный свет (солнце). Area Light — плоский источник (софтбокс). Настройка освещения: Добавьте источник света (Shift + A → Light → выберите тип). Переместите его в сцене (G). Настройте силу (Strength) и цвет в свойствах света. Рендеринг: Перейдите во вкладку Render Properties (иконка камеры). Выберите движок: Eevee (быстрый) или Cycles (фотореалистичный). Настройте разрешение и количество сэмплов. Нажмите Render → Render Image (F12) для создания финального изображения. Урок 6. Экспорт и итоги Экспорт проекта: Модель: File → Export → выберите формат (например, .obj или .fbx). Изображение: после рендера Image → Save As. Итоги курса: Вы научились: устанавливать и настраивать Blender; ориентироваться в интерфейсе; создавать и редактировать 3D‑объекты; применять материалы и текстуры; настраивать освещение и выполнять рендеринг; экспортировать результаты работы. Советы для дальнейшего развития: практикуйтесь ежедневно — создавайте простые сцены; изучайте продвинутые инструменты (скульптинг, анимация, симуляции); присоединяйтесь к сообществам Blender для обмена опытом.
